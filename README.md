# Portfolio Full Stack System

## Structure
- **backend/**: NestJS API Application
- **frontend/**: Angular v18 Application (SSR enabled)

## Development

### Backend
```bash
cd backend
npm run start:dev
```
Runs on http://localhost:3000

### Frontend
```bash
cd frontend
npm run start
```
Runs on http://localhost:4200
